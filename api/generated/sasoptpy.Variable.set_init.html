sasoptpy.Variable.set_init¶

Variable.set_init(self, init=None)[source]¶

Changes initial value of a variable

Parameters
initfloat or None

Initial value of the variable

Examples

>>> x = so.Variable(name='x')
>>> x.set_init(5)

>>> y = so.Variable(name='y', init=3)
>>> y.set_init()
