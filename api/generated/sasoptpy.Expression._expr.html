sasoptpy.Expression._expr¶

Expression._expr()[source]¶

Generates the OPTMODEL compatible string representation of the object.

Examples

>>> f = x + y ** 2
>>> print(f)
x + (y) ** (2)
>>> print(f._expr())
x + (y) ^ (2)
